# Builders MTY

Una comunidad abierta de estudiantes y desarrolladores en Monterrey enfocada en construir software y compartir conocimiento. Iniciada por alumnos de la UANL (FCFM), Builders MTY busca ser un punto de encuentro para quienes disfrutan de programar y crear proyectos.

## 🛠 El Stack

- **Framework:** Next.js 14+ (App Router)
- **Estilos:** Tailwind CSS
- **Backend:** Supabase
- **Deployment:** Railway

## Si quieres destripar esto(no se por que lo harias)

1. **Clona el repositorio:** `git clone https://github.com/Raulgooo/BuildersMTY`
2. **Instala dependencias:** `pnpm install`
3. **Inicia el entorno de desarrollo:** `pnpm dev`
4. **Visita:** `http://localhost:3000`

## Conéctate

- **Discord:** Únete para platicar con otros builders, pedir feedback o colaborar en proyectos.
- **Builders Network:** Un directorio donde mostramos lo que estamos construyendo.
